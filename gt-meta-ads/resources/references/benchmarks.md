# Meta Ads B2B Benchmarks & Metrics

Benchmarks swing hard by vertical, offer, creative quality, and audience temperature. Use as directional gut-checks, never as targets. The metric that matters is whether spend produces pipeline. Focus on cost-per-result trend over time, not absolute numbers. These are industry-reported ranges, not GT-measured.

## Cost benchmarks (2026, B2B)

| Metric | Range / note |
|--------|--------------|
| CPM (cold) | $20-60 for B2B (vs $8-20 B2C) |
| CPL (Instant Form / low-friction) | $15-80 typical |
| CPL (landing page) | Higher than Instant Form, but often better quality |
| Cost per SQL / pipeline | The number that matters; only visible with CRM-synced offline data |
| Retargeting CTR | ~1.2-1.8% |
| Warm LP / form conversion rate | 15-25% |
| Video completion (awareness) | 25%+ strong for B2B |
| Instant Form completion rate | 10-15% healthy |

## Frequency & fatigue

| Signal | Healthy | Warning |
|--------|---------|---------|
| Frequency (cold prospecting, 7 days) | 1.5-2.5 | 3.0-4.0+ |
| CTR vs baseline | stable | drops ~20%+ = refresh |
| CTR decay on repeated creative | gradual | can drop ~40% over 2 weeks |
| High-spend creative fatigue cycle | -- | every 2-4 weeks |

## Creative volume

| Spend level | New variants/week |
|-------------|-------------------|
| $300+/day | 3-5 |
| Lower spend | 1-3, still on a cadence |

- Run 4-5 ads per ad set. Test 2-4 new concepts/week, one variable per test.
- Cheapest iteration: swap the first 3 seconds (hook), keep the body.

## Learning phase

| Signal | Guideline |
|--------|-----------|
| Events to exit learning | ~50 optimisation events / ad set / week |
| Budget floor | CPA x 50 / 7 per day (e.g. $20 CPA -> ~$140/day/ad set) |
| Retargeting ad set minimum | ~$50/day |
| What resets learning | big budget change, audience change, creative-set swap, event change |

## Funnel budget split (typical B2B)

| Layer | Share |
|-------|-------|
| Prospecting (cold, CBO) | 40-50% |
| Consideration (engagers, viewers, visitors) | 30-40% |
| Retargeting (high-intent, ABO) | 20-25% |

## Audiences

| Item | Guideline |
|------|-----------|
| Custom audience minimum | ~1,000 people |
| Lookalike source minimum | 100 (1,000+ performs better) |
| Lookalike start | 1% for conversions; 3-5% for scale/awareness |
| Retargeting window (B2B) | 60-90 days (longer cycle) |
| Sensitive-category audiences | Blocked/auto-rejected (health, financial status) |

## Attribution

- Default 7-day click / 1-day view; consider 28-day click for long B2B cycles. Keep consistent across campaigns.
- iOS limits view-through for off-platform conversions; lean on click + CAPI.
- 2026 delivery (GEM): ad-level CPA is not a reliable creative-quality read; spend allocation is the stronger signal.
- Platform-reported ROAS is directional; CRM pipeline is truth.

## Placements (B2B)

- Facebook Feed and Stories tend to perform best for B2B; avoid Audience Network for lead gen.
- Reels (9:16, 1080x1920, 15-60s, hook in 3s) is the highest-reach placement in 2026.
- 85% watch without sound: captions and mobile-first are non-negotiable.
