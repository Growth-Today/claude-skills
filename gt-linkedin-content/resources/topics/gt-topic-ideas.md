# STATUS values: Trending | Evergreen | Cooling | Seasonal
#   Trending = hot right now, use soon
#   Evergreen = always relevant, no expiry
#   Cooling = was trending, now fading — use soon or skip
#   Seasonal = tied to a time of year

## How to Use This File

When asked for post ideas, topic suggestions, or "what should I post about":
1. Match the request to the right pillar
2. Pick topics marked Trending first, then Evergreen
3. Always pair a topic with the correct funnel stage and formula from the storytelling sub-skill
4. Use your own real numbers from your own records

When asked for a full week of ideas: pick 1 ToF, 3-4 MoF, 1 BoF across different pillars.

## Quick Reference — When Asked "What Should I Post This Week"

Build a week like this using the above:

Monday: Pillar 3 or 4 (MoF — outbound tactic or ICP system reveal)
Tuesday: Pillar 1 (MoF — AI agent or automation build showcase)
Wednesday: Pillar 4 or 5 (MoF — ICP teardown or tool comparison)
Thursday: Pillar 3 (BoF — resource drop with comment gate, if real asset available)
Friday: Pillar 2 (ToF — AI trend explainer with GTM angle)
Sunday: Pillar 1 or 7 (highest effort post — system reveal or journey post)

Always: match the topic to the person posting.
Brigi topics: system builds, operations architecture, marketing systems, GTM Engineer hiring, onboarding, email infra.
Jani topics: sales motion, discovery call insights, deal patterns, people and culture, sales and finance hiring, revenue ops.
