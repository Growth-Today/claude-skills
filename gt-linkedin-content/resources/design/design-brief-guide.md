# Design Brief Guide — Handing a Visual to a Designer

Most people don't design their own LinkedIn visuals — they work with a designer, or a design tool. This guide helps you write a clear brief so the visual comes back on-brand, on-format, and ready to post. It covers *what to specify*, not a specific brand system — plug in your own colors, fonts, and logo.

## Format policy: single image first

The default for a high-performing B2B LinkedIn post is a **single image, 1080×1350px (4:5 portrait)** — it takes the most feed space on mobile and can be boosted as a single-image ad later. Animated infographics (a short MP4 loop or a well-made GIF) also work. Avoid PDF carousels (not boostable as a single-image ad, no alt text) and avoid cheap/generic GIFs.

## The visual headline formula

Every visual needs a headline that stops the scroll. A strong one:
- Contains at least one **number** (a result, a count, a metric)
- Follows the shape: **[Number] + [method/metric] + [outcome]** — e.g. "500 accounts. 7 signals. 0 guessing."
- Has a short subtitle for social proof — e.g. "The system we run for B2B GTM teams"
- Puts visual emphasis (color, weight) on the key number or outcome word

## What every brief should specify

1. **Format & dimensions** — single image, 1080×1350px (4:5).
2. **Headline** — the exact headline text (use the formula above).
3. **Content sections** — the blocks of content, in order, with the exact copy for each. Keep it dense but readable at thumbnail size.
4. **Hierarchy** — what the eye should hit first, second, third.
5. **Brand** — your colors, fonts, and logo placement (keep this in your own brand kit / design tool; don't reinvent per-brief).
6. **Visual elements** — icons, tool logos, arrows, dividers, any screenshot to include.
7. **Mood** — one line, e.g. "clean, operator, confident — not flashy."

## Brief template

```
# Design Brief: [Title]

## Format
Single image, 1080×1350px (4:5 portrait)

## Headline
[Headline text using the number formula]
Subtitle: [social proof line]

## Content sections (in order)
1. [Section name] — [exact copy]
2. [Section name] — [exact copy]
3. [Section name] — [exact copy]

## Hierarchy
First: [ ] · Second: [ ] · Third: [ ]

## Brand
Colors / fonts / logo: [from your brand kit]

## Visual elements
[icons, tool logos, arrows, screenshots]

## Mood
[one line]
```

## Proven single-image formats to brief for

- **Cheat sheet** — everything on one topic on one dense page (roles, tools, metrics, steps). Highest save rate; gets referenced for months.
- **Tool map / stack** — a layer-by-layer view of the tools in a workflow, with logos.
- **Comparison visual** — two columns, "how X is viewed by A vs by B", or before/after.
- **Proof visual** — a real dashboard/tool screenshot with a bold headline overlay. High credibility because the numbers are real.

## Animated infographic notes

If you brief an animated version: a short MP4 loop (or a clean GIF), 1.5–2x speed, seamless loop, under ~15 seconds. Never a throwaway low-effort GIF — it reads as off-brand.

## Format selection always starts from the content

Pick the format from the post, in this order: **content topic → content pillar → goal (ToF reach / MoF engagement / BoF pipeline)**. Then brief the matching format above.
